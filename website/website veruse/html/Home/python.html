How to download Python

តើ Python គឺជាអ្វី? ហើយវាមានសារប្រយោជន៍អ្វីខ្លះ?

ជា​ភាសា​សរសេរ​កម្មវិធី​កូដ​បើក​ចំហ (open source programming language) ដែល​ត្រូវ​បាន​បង្កើត​ឡើង​ឲ្យ​មាន​ភាព​ល្អ និង​ងាយ​ស្រួល​ក្នុង​ការ​អាន ។ វា​ត្រូវ​បាន​បង្កើត​ឡើង​ដោយ​អ្នក​សរសេរ​កម្មវិធី​ម្នាក់សញ្ជាតិហូឡង់ដែល​មាន​ឈ្មោះថា ហ្គីដូ វ៉ាន រ៉ូសសឹម (Guido van Rossum) ក្នុង​ឆ្នាំ ១៩៩១ ។

1.អ្នកអាចប្រើ Python សម្រាប់អ្វីៗជាច្រើន ដូចជា កម្មវិធីវេប វិភាគទិន្នន័យ បញ្ញាធិបតី និងការបង្ហាញទិន្នន័យ។

2.Python មានសន្ទស្សន៍ដែលអាចយល់បានយ៉ាងងាយស្រួល ដែលជួយឲ្យកម្មវិធីទាំងអស់សរសេរកូដបានយ៉ាងច្បាស់និងមានប្រសិទ្ធភាព។

3.មានបណ្ណាល័យនិងប្រព័ន្ធផ្សេងៗដែលបន្ថែមអនុសាសន៍សម្រាប់ Python ដែលធ្វើអោយវាអាចអនុវត្តការងារផ្សេងៗបានកាន់តែមានសមត្ថភាព។

4.Python អាចដំណើរការលើប្រព័ន្ធប្រតិបត្តិការ ដូចជា Windows, macOS, and Linux។
