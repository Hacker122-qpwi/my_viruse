How to download Visual Studio

តើ Visual Studio Code គឺជាអ្វី?ហើយវាមានសារប្រយោជន៍អ្វីខ្លះ?

VS Code (Visual Studio Code) គឺជាកម្មវិធីកែសម្រួលកូដដែលឥតគិតថ្លៃ និងបើកប្រភពដោយ Microsoft។ វាគឺជាកម្មវិធីដែលអ្នកអភិវឌ្ឍន៍ប្រើសម្រាប់សរសេរ និងកែសម្រួលកូដសម្រាប់ភាសាកម្មវិធីនានា និងជាស្ថាប័ន។ VS Code មានសមត្ថភាពផ្ទាល់ខ្លួនខ្ពស់ និងផ្តល់នូវលក្ខណៈពិសេសច្រើនដែលធ្វើឲ្យការសរសេរកូដមានសមត្ថភាពល្អប្រសើរ។

1.VS Code បញ្ជាក់ពណ៌ទៅលើកូដរបស់អ្នកយ៉ាងហោចណាស់បំផុត ដើម្បីធ្វើឲ្យវាអានបានងាយ និងយល់ដឹងបានល្អជាង នៅពេលដែលមានគម្រោងធំ។

2.លក្ខណៈពិសេសនេះផ្តល់ការណែនាំកូដ និងការបំពេញឯកសារ ពេលអ្នកវាយសរសេរ ជួយសំរបសំរួលការសរសេរកូដរបស់អ្នកឲ្យមានល្បឿនលឿន។

3.អ្នកអាចបើក terminal ក្នុង VS Code ដើម្បីប្រើប្រាស់ពេលអនុវត្តពាក្យបញ្ជា (ដូចជា ពាក្យបញ្ជា Git ការរត់ម៉ាស៊ីនបម្រើ និងផ្សេងៗ) ដោយគ្មានការផ្លាស់ប្តូរទៅកាន់វីនដោចផ្សេង។

4.VS Code អនុញ្ញាតឲ្យអ្នកតំឡើង extensions ដើម្បីបន្ថែមមុខងារ។ អ្នកអាចតំឡើង extension សម្រាប់:

  • - UPPORT សម្រាប់ភាសាកម្មវិធីបន្ថែម (Python, Java, C++ និងផ្សេងៗ)

  • - ស្បែក និងថតរូបផ្ទាំង (Themes and icon packs) ដើម្បីផ្លាស់ប្តូរពណ៌សរូបភាព

  • - ការតភ្ជាប់ Git សម្រាប់ការគ្រប់គ្រងទំព័រភាសា

  • - Linter និង Formatter ដើម្បីពិនិត្យ និងសម្រួលស្ទីលកូដ

  • - Live Server (ដើម្បីរត់ទំព័របណ្ដាញរបស់អ្នកនៅក្នុងកម្មវិធីរុករកសម្រាប់ការអាប់ដេតវ៉ិបប្រសិនបើមានការផ្លាស់ប្តូរ)

5.VS Code មានឧបករណ៍ដើម្បីពិនិត្យកូដ ដូចជា ការកំណត់ breakpoints និងការត្រួតពិនិត្យតម្លៃជាក់លាក់ក្នុងពេលពិត។

6.វាជួយការការប្រើប្រាស់ Git ដែលអនុញ្ញាតឲ្យអ្នកចម្លងធនធានរ៉េពូស៊ីទោរ ធ្វើការបញ្ជាក់ផ្លាស់ប្តូរ បង្កើតសាខា និងការប្រតិបត្តិការផ្សេងៗតាមផ្លូវក្នុងកម្មវិធី។

7.VS Code គាំទ្រភាសាច្រើនពីដើមហើយ អ្នកអាចបន្ថែមភាសាបន្ថែមតាមរយៈ extensions។

8.មានស្រាប់សម្រាប់ប្រព័ន្ធប្រតិបត្ដិការ Windows, macOS និង Linux។

វិធីប្រើ Visual Studio Code

1.អ្នកអាចទាញយក VS Code ពីគេហទំព័រដើម របស់វា។

2.បន្ទាប់ពីតំឡើង VS Code អ្នកអាចបើកមើល Extensions (Ctrl+Shift+X) ដើម្បីបន្ថែមមុខងារផ្សេងៗ។

3.បើកប្លុកឬឯកសារអ្វីមួយ និងចាប់ផ្តើមសរសេរកូដ។
